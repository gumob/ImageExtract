ImageExtract

public class ImageExtract

The class that allows you to get the size of a remote image without downloading.

  • extract(_:chunkSize:downloadOnFailure:)

    A function to get the size of a remote image synchronously.

    Declaration

    Swift

    public class func extract(_ request: ImageRequestConvertible,
                          chunkSize: ImageChunkSize = .small,
                          downloadOnFailure: Bool = false) -> CGSize

    Parameters

    request

    An image url to request. String, URL, and URLRequest are conform to ImageRequestConvertible protocol.
    chunkSize

    Chunk size to download. The default value is ImageChunkSize.small. (100 bytes)
    downloadOnFailure

    A Boolean value indicating whether all data including bitmap should be downloaded if it fails to extract an image size from chunk data. The default value is false.

    Return Value

    A size of an image.
